# Quote Generator

A web application that generates inspirational quotes based on user-selected topics.

## 🌐 Live Demo
**Deployment:** [https://nexium-wajahat-assignment1-gwpl.vercel.app/](https://nexium-wajahat-assignment1-gwpl.vercel.app/)

## 🚀 Features

- **Quote Generation**: Search and display inspirational quotes by topic (motivation, success, love, etc.)
- **Responsive Design**: Works on desktop and mobile devices
- **Copy to Clipboard**: Copy quotes with one click
- **Social Sharing**: Share quotes on WhatsApp, Facebook, Twitter, Telegram, and Instagram
- **Animated UI**: Floating bubbles and sparkles for visual appeal
- **Modern Design**: Built with shadcn/ui components and Tailwind CSS
